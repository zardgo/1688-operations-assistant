# 05 保温杯类目知识库

这部分是软件的差异化资产。通用 1688 工具只能告诉员工“优化商品”“提升转化”，保温杯工具必须知道买家采购保温杯时真正关心什么。

## 保温杯采购关注点

| 关注点 | 买家实际问题 | 商品/客服必须回答 |
| --- | --- | --- |
| 材质 | 是 304 还是 316？是否食品接触安全？ | 材质、内胆、检测或说明。 |
| 容量 | 350ml、500ml、750ml、1000ml 哪个适合？ | 容量、尺寸、适用人群。 |
| 保温时长 | 能保温多久？是否夸大？ | 测试条件和保温说明。 |
| 杯盖结构 | 会不会漏水？单手开合吗？ | 杯盖结构、密封方式、使用场景。 |
| 定制 Logo | 能不能印 Logo？起订量多少？ | 工艺、MOQ、打样费、打样周期。 |
| 拿样 | 能不能先拿样？样品多少钱？ | 样品政策、样品发货时间。 |
| 交期 | 现货多久发？定制多久发？ | 现货交期、定制交期、确认稿节点。 |
| 包装 | 有礼盒吗？装箱数量多少？ | 包装方式、装箱规格、礼品方案。 |
| 开票 | 能不能开票？含税价多少？ | 开票类型、含税/不含税报价。 |
| 现货 | 是否真实有库存？ | 库存、颜色、容量、发货承诺。 |

## 商品信任清单

主推商品至少要检查：

- 是否写清材质。
- 是否写清容量。
- 是否写清保温时长。
- 是否写清杯盖和密封结构。
- 是否支持拿样。
- 是否支持 Logo 定制。
- 是否写清起订量。
- 是否写清交期。
- 是否写清包装。
- 是否支持开票。
- 是否展示真实场景图。
- 是否有检测或食品接触相关说明。

当商品有访客但询盘低，优先检查这张清单。

## 信任信号

以下信号可以作为保温杯商品和店铺表达的检查方向，但不使用本地样本中的次数、占比或中位数作为目标依据。

| 信号 | 经营含义 |
| --- | --- |
| 产地和工厂能力 | 店铺表达要讲清产地、设备、产能、质检和供货稳定性。 |
| 工厂身份 | 平台工厂身份可作为信任背书，但不能替代真实履约能力。 |
| 可开专票 | B2B 买家重视合规和对公交易。 |
| 来图/来样加工 | 定制能力是保温杯成交核心之一。 |
| 支持拿样 | 拿样降低首次采购风险。 |
| 认证和检测 | 资质和品质说明影响信任，但必须真实。 |

注意：

```text
金冠品已过期，不能作为长期商品成长目标。
金牌制造是工厂身份和平台背书，仍可作为对标信号。
```

## 询盘高频问题

客服快捷回复至少覆盖：

1. 最小起订量多少。
2. 能不能定制 Logo。
3. 打样多少钱。
4. 打样和大货交期多久。
5. 有没有现货。
6. 能不能开发票。
7. 是 304 还是 316。
8. 保温多久。
9. 有没有礼盒包装。
10. 运费怎么算。
11. 大货是否支持阶梯价。
12. 是否能先拿样再下大货。

## 商品角色分层

| 角色 | 作用 | 主要指标 |
| --- | --- | --- |
| 引流款 | 拿曝光和访客。 | 曝光、访客、点击。 |
| 利润款 | 承接有质量询盘。 | 毛利率、询盘转化。 |
| 定制款 | 拿 Logo 定制和找工厂积分。 | 定制询盘、合约支付、交易积分。 |
| 礼品款 | 做团购、企业采购、节日福利。 | 客单价、包装需求、交期成功率。 |
| 复购款 | 老客补货和渠道采购。 | 老客回头率、复购周期。 |
| 风险款 | 售后、退款、质量问题集中。 | 品质退款率、差评、售后率。 |
| 清库存款 | 库存高、转化弱，需要回款。 | 库存、折扣、成交速度。 |

## 类目风险

| 风险 | 表现 | 应对 |
| --- | --- | --- |
| 材质承诺过度 | 买家收到后质疑 304/316。 | 材质说明谨慎，保留检测/供应链依据。 |
| 保温时长夸大 | 差评、退款、品质问题。 | 写清测试条件，不做过度承诺。 |
| 定制交期失控 | 履约率下降、平台介入。 | 确认稿、打样、生产、包装节点留痕。 |
| 低价引流亏损 | GMV 增长但毛利下降。 | 单品毛利表和广告准入规则。 |
| 样品客户低质量 | 客服时间和样品成本被消耗。 | 样品客户分层和回访机制。 |
