# 新灯塔5.0服务体验指标

> 重要：本文来自用户提供的结构化资料，并已补充新灯塔 AI 页面作为数据源。该页面可用于录入新灯塔和店铺服务星级数据；规则原文、类目口径和权重仍需后续补齐。

## 1. 规则定位

新灯塔是平台衡量商家全链路买家采购体验的服务指标体系。它不只是客服或物流指标，而是会影响：

- 店铺星级；
- 搜索加权；
- 活动报名资格；
- L等级升级；
- 严选、哇噢定制、找工厂等高价值场景的承接能力。

对 1688 运营助手来说，新灯塔应该归入“服务履约模块”，并和 L 等级、生意积分、交易勋章、商品采购指数共同构成经营目标基座。

## 2. 计分结构

| 维度 | 权重 | 核心指标 | 保温杯/消费品优先关注 |
| --- | ---: | --- | --- |
| 物流体验分 | 35% | 48小时揽收率、物流时效达成率、发货物流退款率 | 是 |
| 售后体验分 | 30% | 退款处理时长、平台介入率 | 是 |
| 咨询体验分 | 20% | 旺旺3分钟响应率、咨询满意度 | 是 |
| 商品体验分 | 15% | 品质退款率、品质问题率 | 是 |
| 特色保障服务分 | 附加/加分项 | 发货保障订单占比、在线运踪订单占比 | 是 |

工业品路径中，物流体验会更强调履约率。保温杯店铺默认按消费品路径，但如果后台主营行业被系统判定为其他类目，应以后台展示为准。

## 4. 与其他目标的关系

| 关联目标 | 关系 |
| --- | --- |
| L等级 | L4/L5 相关健康度指标会受到咨询、履约、售后指标影响；具体门槛仍需后台确认。 |
| 商品采购指数4.0 | 商品服务、品质、履约表现会影响采购指数相关诊断和权益承接。 |
| 找工厂 | 找工厂询盘和定制订单需要稳定响应与履约，否则成交越多，服务风险越高。 |
| 严选 | 严选商品需要更强的服务、品质和渠道承接能力，新灯塔低分会削弱权益兑现。 |
| 交易勋章 | 交易额增长若伴随退款、纠纷、物流异常，会损害长期等级和经营安全。 |

## 5. 软件中的使用方式

新灯塔不适合只做一个“总分看板”。更适合拆成三层：

1. 经理视角：看服务体验是否卡住 L 等级、活动报名、搜索扶持。
2. 运营视角：看哪个维度拖分，以及它对应什么业务动作。
3. 员工视角：只看到当天要处理的 checklist，例如“未及时回复咨询”“待处理退款”“物流异常订单”“高风险品质反馈”。

系统不应要求员工写长篇动作证据。更合理的证据方式是：

- checklist 勾选；
- 自动抓取后台状态；
- 上传截图或表格；
- 异常项才要求文字说明。

## 5.1 录入频率

新灯塔和店铺服务星级适合作为趋势数据，不适合让员工每天手工填一堆字段。

| 频率 | 做什么 | 目的 |
| --- | --- | --- |
| 每日 | 只录入总分、店铺服务星级、最低分维度和明显异常项 | 生成今日服务风险 checklist |
| 每周 | 补全物流、售后、咨询、商品体验分和关键明细指标 | 做趋势 BI 和卡点复盘 |
| 异常触发 | 星级下降、某维度大幅下降、退款/物流异常上升时补截图 | 追原因和留证据 |

## 6. 保温杯店铺优先级

对保温杯这种消费品店铺，优先级建议为：

1. 旺旺3分钟响应率：直接影响咨询体验，也影响询盘转化。
2. 48小时揽收率：低成本、可标准化、对服务分影响大。
3. 退款处理时长：减少平台介入和差评扩散。
4. 品质退款率：保温杯容易涉及漏水、异味、磕碰、容量不符、定制瑕疵。
5. 发货保障/在线运踪：既能提升买家信任，也能降低售后解释成本。
