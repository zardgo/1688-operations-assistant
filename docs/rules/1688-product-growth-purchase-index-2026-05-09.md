## 2. 核心变化

| 变化 | 旧体系 | 新体系 | 软件影响 |
| --- | --- | --- | --- |
| 商品成长体系 | 金冠品、潜力品、多套成长标准 | 不达标商品、潜力商品、采购指数4.0商品 | 不能继续把“金冠品数”作为长期核心指标。 |
| 新品成长 | 新品成长积分制 | 新品采购指数分 | 新品不再按成长积分判断，应看新品采购指数。 |
| 达标门槛 | 金冠规则、成长分规则 | 采购指数 >= 4.0 | 商品成长目标统一改为采购指数4.0达标商品数。 |
| 评价维度 | 消费品/工业品不同逻辑 | 口碑、价格、服务、品质、商誉 | 诊断动作要覆盖价格、服务、品质、商誉。 |
| 镇店之宝 | 金冠商品可配置 | 采购指数 >= 4.0 商品可配置 | 镇店之宝动作必须改用采购指数门槛。 |
| 权益 | 金冠品权益 | 搜索/推荐、核心渠道、站外渠道等采购指数权益 | 商品成长达标会影响流量和渠道扶持。 |

## 3. 商品分层标准

| 新等级 | 判断规则 | 软件字段建议 | 经营含义 |
| --- | --- | --- | --- |
| 不达标商品 | 采购指数 = 0 | `purchase_index_score = 0` | 商品基础服务、品质、价格或商誉存在明显短板。 |
| 潜力商品 | 0 < 采购指数 < 4.0 | `0 < purchase_index_score < 4.0` | 有成长机会，但未达到平台扶持门槛。 |
| 采购指数4.0商品 | 采购指数 >= 4.0 | `purchase_index_score >= 4.0` | 达到商品扶持基础门槛，可获得搜索/推荐和渠道扶持。 |

## 6. 权益体系

采购指数4.0是商品扶持基础门槛。若商品加入 1688 严选且符合采购指数4.0，可获得更高优先级权益。

| 权益类型 | 采购指数4.0普通商品 | 严选且采购指数4.0商品 |
| --- | --- | --- |
| AI 诊断 | AI 诊断 | AI 诊断高阶版 |
| 流量权益 | 搜索/推荐扶持 | 搜索/推荐加权优先扶持 |
| 商品表达 | 商品核心卖点表达 | 严选标、严选采购指数、专属商品详情表达 |
| 站外渠道 | 支付宝、跨境、分销、即时零售等渠道扶持 | 同类渠道优先扶持 |
| 专属场域 | 无 | 严选频道、严选榜单、严选搜索 TAB、会员店 |
| 核心渠道 | 新人专区、天天特卖扶持 | 新人专区、天天特卖优先扶持 |
| 营销活动 | S 级活动、A++ 活动扶持 | 严选专属活动、会员专属营销、百亿补贴 |
| 镇店之宝 | 达成采购指数4.0可设置 | 达成严选采购指数4.0可设置，更多配额 |

## 9. 对保温杯店铺的经营含义

保温杯商品成长不再只是“补金冠品数”，而是要围绕采购指数五维度做商品经营。

| 维度 | 保温杯动作方向 |
| --- | --- |
| 口碑 | 控制漏水、掉漆、保温不达预期、容量偏差等差评和退款。 |
| 价格 | 建立试单款、主推款、定制大货款三档报价，避免低价亏损或价格无竞争力。 |
| 服务 | 提升响应率、交期确定性、拿样/定制答复效率。 |
| 品质 | 写清 304/316 材质、保温时长、杯盖结构、包装和检测说明。 |
| 商誉 | 降低退款、介入、违规和服务体验问题。 |
