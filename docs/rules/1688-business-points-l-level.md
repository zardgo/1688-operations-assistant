# 1688 生意积分与 L 等级规则卡

> 来源：用户提供的「1688 平台最新商家成长体系（L 等级）/ 生意积分」结构化说明。  
> 整理日期：2026-05-24  
> 状态：`source_found`。缺官方链接、发布日期和后台截图；软件可先用于目标拆解，但应显示来源待补，并允许后台口径覆盖。  
> 适用：保温杯按消费品商家体系候选，最终以商家工作台主营行业归属为准。

## 1. 规则边界

生意积分是衡量商家经营质量与平台价值贡献的指标，用于驱动 L 等级升级、匹配流量权益和专属服务。

它不等同于：

- 交易勋章。
- 找工厂牌级。
- 采购指数4.0。
- 严选采购指数。
- 服务体验星级。

软件里应把它放在「店铺成长目标」模块，而不是商品成长或找工厂模块。

## 2. 统计周期与数据边界

| 项目 | 口径 |
| --- | --- |
| 统计周期 | 近 30 天累计数据。 |
| 数据处理 | 剔除违规及退款数据。 |
| 计算方式 | 系统自动计算，不可手动触发。 |
| 数据来源 | 商家工作台、生意参谋、数字营销推广报表。 |
| 行业体系 | 消费品商家、工业品商家两套体系。 |

## 3. 消费品商家生意积分

适用于服饰、家居、食品等消费品方向。保温杯店铺先按消费品路径处理，具体以后台主营行业为准。

```text
消费品商家生意积分 = 买家数积分 + 交易额积分 + 生意加速分
```

| 组成 | 规则 | 软件字段 |
| --- | --- | --- |
| 买家数积分 | 1 个有效买家 = 1 分；有效买家为支付成功买家人数，同生意参谋口径。 | `valid_buyer_count_30d` |
| 交易额积分 | 1000 元交易额 = 1 分。 | `gmv_30d` |
| 数字营销买家加速分 | 每 1 个数字营销买家额外 +3 分。 | `digital_marketing_buyer_count_30d` |
| 场景交易额加速分 | 大严选、托管、哇噢定制等场景交易额，每 1000 元额外 +3 分。 | `accelerated_scene_gmv_30d` |
| 优质供应商加速分 | 超级工厂、深度验厂等优质供应商一次性 +600 分。 | `premium_supplier_bonus` |

## 4. 工业品商家生意积分

适用于交通运输、机械、五金、仪器仪表等工业品方向。保温杯默认不走工业品体系，除非后台主营行业明确归入工业品。

```text
工业品商家生意积分 = 询盘数积分 + 交易额积分 + 生意加速分
```

| 组成 | 规则 | 软件字段 |
| --- | --- | --- |
| 询盘数积分 | 1 个有效询盘 = 0.3 分；含旺旺、找工厂询价单、电话等，每人每天计 1 次。 | `valid_inquiry_count_30d` |
| 交易额积分 | 1000 元交易额 = 1 分。 | `gmv_30d` |
| 数字营销询盘加速分 | 每 1 个数字营销询盘额外 +0.9 分。 | `digital_marketing_inquiry_count_30d` |
| 场景交易额加速分 | 大严选、托管、行家选等场景交易额，每 1000 元额外 +3 分。 | `accelerated_scene_gmv_30d` |
| 深度验商/验厂加速分 | 深度验商或验厂 +200 分。 | `verified_supplier_bonus` |
| 签约行家选加速分 | 签约行家选 +50 分。 | `expert_selection_bonus` |

## 6. L 等级升级路径候选

> 来源：用户提供的 2026 年《商家 L 等级成长体系规则》结构化说明。  
> 状态：待官方链接、发布日期、后台截图确认。  
> 使用限制：确认前只能作为目标候选和字段设计参考，不能作为软件硬判断。

L 等级共 L1-L6 六档，采用月度动态评估、自动升降、权益绑定机制。核心逻辑是以生意积分为基础门槛，叠加多维健康度指标。

| 等级 | 生意积分门槛（近 30 天） | 健康度要求候选 | 软件字段 |
| --- | --- | --- | --- |
| L1 | >= 0 分，新店默认 | 无强制要求 | `l_level` |
| L2 | >= 300 分 | 店铺基础信息完整率 >= 95%；无严重违规 | `shop_profile_completeness_rate`、`serious_violation_count` |
| L3 | >= 1200 分 | 买家满意度 >= 85%；发货及时率 >= 90% | `buyer_satisfaction_rate`、`shipping_timeliness_rate` |
| L4 | >= 3000 分 | 询盘响应率 >= 95%；48 小时内回复率 >= 90% | `inquiry_response_rate`、`reply_rate_48h` |
| L5 | >= 6500 分 | 交易纠纷率 <= 1.2%；定制类交易占比 >= 15% | `trade_dispute_rate`、`custom_trade_ratio` |
| L6 | >= 12000 分 | L5 持续满 3 个月；年度 GMV >= 500 万元；有深度验厂或超级工厂认证 | `l5_continuous_months`、`annual_gmv`、`deep_factory_verified`、`is_super_factory` |

定制类交易占比候选口径：

```text
定制类交易占比 = 哇噢定制 + 找工厂合约订单交易额 / 总交易额
```

实际落库时应拆成：

```text
custom_trade_ratio = (wow_custom_gmv_30d + factory_contract_gmv_30d) / gmv_30d
```

所有 L 等级数据候选来源为「生意参谋-成长中心」，T+1 日更新。实际字段名和更新时间以后台展示为准。

## 7. L 等级权益候选

> 来源同上，待官方确认。权益名称、百分比、流量包数量、开放条件可能随平台调整，软件只能在规则确认后展示为硬权益。

| 权益类型 | L3 候选权益 | L4 候选权益 | L5/L6 候选权益 |
| --- | --- | --- | --- |
| 搜索加权 | 曝光权重提升 | 更高曝光权重 | 更高曝光权重，首屏 L5/L6 标识 |
| 活动报名 | 可报日常活动 | 可报行业大促 | 优先审核，活动资源位保量 |
| 流量扶持 | 免费猜你喜欢基础曝光 | 推荐流量包 | 定制化流量包，如跨境买家定向推送、定制买家专属频道 |
| 服务能力 | 可开通七天无理由、破损包赔等服务 | 可开通极速开票、官方直送等服务 | 可开通哇噢定制专属服务、超级工厂专属客服通道 |
| 数据工具 | 基础生意参谋 | 客户分层分析、竞品对标 | L 等级专属诊断报告、AI 经营建议引擎 |

注意：权益应和「商家身份、类目、会员类型、后台开通条件」共同判断，不能只看 L 等级。

## 8. 诊断规则

| 触发条件 | 诊断 | 今日 checklist |
| --- | --- | --- |
| 消费品买家数不足 | 新客成交不足，L 等级升级慢 | 提升访客询盘率、询盘支付率，做小单/样品后转正式单。 |
| 消费品交易额不足 | 客单价或成交规模不足 | 优化主推款组合、阶梯价、大货报价、老客补货。 |
| 有广告花费但数字营销买家不足 | 投放没有形成有效买家 | 检查关键词、人群、落地商品、询盘质量和报价承接。 |
| 场景交易额不足 | 未利用严选、托管、哇噢定制等加速场景 | 检查主推商品是否适合进入严选/哇噢定制/托管场景。 |
| 工业品询盘积分被误用于保温杯 | 行业体系误判 | 阻断，要求确认后台主营行业。 |
| 生意积分上升但毛利下降 | L 等级增长可能吞利润 | 先查毛利、退款、履约，再决定是否继续冲。 |
| L 等级基础积分达标但健康度不达标 | 不能升级或可能降级 | 优先修健康度短板，不继续盲目冲交易额。 |
| L5 缺定制类交易占比 | 定制场景承接不足 | 检查哇噢定制、找工厂合约订单、定制商品入口和定制报价 SOP。 |
| L4 缺响应指标 | 客服承接能力不足 | 查未回复、慢回复、分流、快捷回复和 AI 客服答案质量。 |
| L3 缺满意度或发货及时率 | 买家体验和履约弱 | 查差评、咨询满意度、发货延期、物流揽收。 |

## 9. 与其他模块的关系

| 模块 | 关系 |
| --- | --- |
| 交易勋章 | 都使用近 30 天交易数据，但交易勋章有独立金额/买家数门槛和权益。 |
| 采购指数4.0 | 商品成长目标；可以间接影响 L 等级，但不是同一积分体系。 |
| 严选采购指数 | 严选商品成长目标；严选场景交易额可能带来生意加速分。 |
| 找工厂 | 哇噢定制和定制交易可能影响加速场景，但找工厂牌级不是 L 等级。 |
| 数字营销 | 数字营销买家/询盘可能带来额外加速分，但必须看广告费率和毛利。 |

## 10. 保温杯落地

保温杯店铺应默认按消费品商家生意积分处理：

```text
有效买家数 + 交易额 + 数字营销买家 + 严选/托管/哇噢定制场景交易额 + 优质供应商加分
```

优先级建议：

```text
毛利安全 > 服务履约 > 有效买家数 > 交易额 > 生意积分 > L等级权益
```

不要为了 L 等级冲交易额而接低毛利、高售后、高履约风险订单。
