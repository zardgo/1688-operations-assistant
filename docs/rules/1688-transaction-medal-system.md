## 1. 规则边界

交易勋章是交易规模和买家数相关的成长体系，不等同于：

- 店铺 L 等级。
- 找工厂牌级。
- 采购指数4.0。
- 严选采购指数。

软件中应单独作为「交易规模目标」模块。它可以影响权益和镇店之宝配额，但不能压过毛利、履约、售后和虚假交易红线。

## 2. 适用体系

| 体系 | 适用范围 | 保温杯相关判断 |
| --- | --- | --- |
| 体系一 | 消费品 25 个类目及包装类目，包括居家日用品、日用餐厨饮具等 | 保温杯大概率适用，需后台确认 |
| 体系二 | 暂无行业适用 | 暂不使用 |
| 体系三 | 工业品、原材料及相关行业 | 保温杯不按工业品处理，除后台另有归类 |

适用行业按卖家商铺中商品数最多的行业判断。该规则适用于阿里巴巴大市场所有频道及页面。

## 3. 体系一等级门槛

| 等级 | 晋级要求 | 软件字段 |
| --- | --- | --- |
| 1A | 近 30 天累计勋章金额 > 0 元 | `medal_amount_30d` |
| 2A | 近 30 天累计勋章金额 >= 1 万元 | `medal_amount_30d` |
| 3A | 近 30 天累计勋章金额 >= 10 万元；近 30 天买家数 >= 20 个；诚信通会员 | `medal_amount_30d`、`medal_buyer_count_30d`、`is_chengxintong_member` |
| 4A | 近 30 天累计勋章金额 >= 30 万元；近 30 天买家数 >= 30 个；诚信通会员 | 同上 |
| 5A | 近 30 天累计勋章金额 >= 100 万元；近 30 天买家数 >= 40 个；诚信通会员 | 同上 |

## 4. 数据口径

| 指标 | 口径 |
| --- | --- |
| 近 30 天累计勋章金额 | 基于商家近 30 天交易额，剔除虚假交易、退款部分、样品订单；系统可能按节假日动态调整。 |
| 近 30 天累计买家数 | 剔除虚假交易、退款部分、样品订单。 |
| 数据计算方式 | T+1 计算。例：8 月 2 日计算近 30 天，表示 7 月 3 日到 8 月 1 日交易数据。 |
| 更新周期 | 每天计算一次，每天 24 点统一调整、更新一次。 |
| 升级方式 | 可跃级升级。 |
| 降级方式 | 逐级降级，不可多级降。 |
| 违规处理 | 被判定虚假交易、涉嫌虚假交易或贸易纠纷全责，一律清除违规交易数据。 |

## 5. 权益摘要

### 诚信通会员权益

| 等级 | 权益 |
| --- | --- |
| 1A | 搜索排名递增；诚易保极速到账服务；融易收大额资金收款服务；普通主播身份；镇店之宝商品配额 1。 |
| 2A | 搜索排名递增；诚易保费率更低；融易收；达人主播身份；镇店之宝商品配额 2。 |
| 3A | 搜索排名递增；诚易保费率 0.15%；荣耀主播；支付宝货源市场；跨境伙拼；淘货源频道搜索及场景排序加权；微供垂直频道推商橱窗位；镇店之宝商品配额 3。 |
| 4A | 诚易保费率 0.12%；镇店之宝商品配额 4；其他权益同高等级交易勋章权益。 |
| 5A | 诚易保费率 0.1%；镇店之宝商品配额 5；其他权益同高等级交易勋章权益。 |

### 实力商家会员权益

| 权益类别 | 内容 |

| 品牌展现 | 实力身份、全景拍摄、专属 work 后台等。 |

| 专属服务 | 专享培训、金融、服务等；诚易保费率最低可至 0.07%。 |

## 7. 诊断规则

| 触发条件 | 诊断 | 今日 checklist |
| --- | --- | --- |
| 金额达标但买家数不足 | 交易集中在少数买家，升级受买家数限制 | 做新客询盘转化、补小单成交、开发渠道买家。 |
| 买家数达标但金额不足 | 客单价或成交规模不足 | 推主推款组合、阶梯价、大货/复购方案。 |
| 接近 3A/4A/5A | 交易勋章权益即将提升 | 先检查毛利、履约、售后，再决定是否冲刺。 |
| 退款或样品单占比高 | 勋章金额可能不计入或被扣除 | 区分样品单、正式单、退款单，避免误判。 |
| 出现虚假交易风险 | 违规交易数据会被清除 | 阻断补单/刷单类动作，生成合规提醒。 |

## 8. 与其他模块的关系

| 模块 | 关系 |
| --- | --- |
| 镇店之宝 | 交易勋章等级可影响商品配额，但 2026-05-09 后商品可配置门槛应看采购指数 >= 4.0。 |
| 店铺 L 等级 | 都和交易增长相关，但不是同一体系。 |
| 采购指数4.0 | 商品资产目标，不等同于交易勋章金额。 |
| 找工厂牌级 | 定制场景身份目标，不等同于交易勋章。 |
| 毛利模块 | 冲交易勋章前必须确认毛利底线。 |

## 9. 保温杯落地

保温杯店铺可以把交易勋章作为中期交易规模目标，但不应直接追 GMV。

推荐判断顺序：

```text
毛利安全 > 履约安全 > 售后风险可控 > 买家数增长 > 勋章金额增长 > 勋章权益
```

员工执行时，交易勋章只回答「交易规模缺口」，不能直接生成低毛利冲量、虚假交易、过度补单等动作。
